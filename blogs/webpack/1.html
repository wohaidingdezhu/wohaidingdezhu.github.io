# 前言

webpack 大部分听起来就感觉很恐怖，并且大部分同学平时也难以接触到，一般是使用大型项目需要用原生 js 去开发,他是资源打包工具。 webpack 四个核心概念 ：入口 entry 出口 output loader 转换器 Plugins 插件

底层：Node.js JS 运行环境

配置按需加载: 路由懒加载 注释语法 webpackchunkName 图片懒加载 vue-lazy 组件库按需加载 第三库插件也需要按需加载

  mode: "development",
  entry: resolve(__dirname, "src/app.js"),
  output: {
    path: resolve(__dirname, "build"),
    filename: "app.js",

  devtool: "source-map",
  module: {
    rules: [
      {
        test: /\.tpl$/,

          "babel-loader",

            loader: "./loaders/tpl-loader",

              log: true,
            },
          },

  },
  plugins: [
    new HtmlWebpackPlugin({
      template: resolve(__dirname, "index.html"),
    }),
  ],
  devServer: {
    port: "32124",

};

  // 去掉空格和换行

  );
  const _log = log
    ? `console.log('compiled the file which is form ${this.resourcePath}')`
    : "";
  return `
        export default (options) => {
            ${tplReplace.toString()}
